Mayorista.md

### Estados Posibles del Mayorista

- **M0 — Visitante:** no tiene cuenta de mayorista.
- **M1 — Mayorista registrado:** cuenta creada, perfil incompleto.
- **M2 — Mayorista activo:** perfil completo y al menos un producto publicado.
- **M3 — Mayorista validado:** tiene ventas realizadas o impacto verificado.

## Como Mayorista

### Descubrimiento / Confianza

- Como mayorista, quiero conocer el propósito del proyecto para saber si se alinea con mis valores e impacto ambiental.  
  → **M0, M1, M2, M3**

- Como mayorista, quiero ver historias y logros de otros productores para confiar en la plataforma.  
  → **M0, M1, M2, M3**

- Como mayorista, quiero acceder a información de contacto para resolver dudas antes de registrarme.  
  → **M0, M1, M2, M3**

---

### Registro y Perfil

- Como mayorista, quiero registrarme en la plataforma para poder ofrecer mis productos.  
  → **M0 (pasa a M1)**

- Como mayorista, quiero completar mi perfil (historia, misión, impacto ambiental) para presentarme a los clientes.  
  → **M1, M2, M3**

- Como mayorista, quiero editar mis datos de perfil para mantener mi información actualizada.  
  → **M1, M2, M3**

---

### Gestión de Productos

- Como mayorista, quiero crear productos (ej. bombas de semillas) para venderlos en la plataforma.  
  → **M1 (pasa a M2)**

- Como mayorista, quiero agregar precio, fotos, descripción, métodos de entrega y stock a mis productos para que los clientes puedan evaluarlos.  
  → **M2, M3**

- Como mayorista, quiero definir opciones de personalización del producto (tipo de flores, cantidad, presentación) para adaptarme a distintos clientes.  
  → **M2, M3**

- Como mayorista, quiero pausar o eliminar productos cuando no tenga stock disponible.  
  → **M2, M3**

---

### Impacto Ambiental

- Como mayorista, quiero mostrar el impacto ambiental de mis productos para generar confianza en los clientes.  
  → **M1, M2, M3**

- Como mayorista, quiero actualizar métricas de impacto (semillas plantadas, zonas intervenidas, proyectos apoyados).  
  → **M2, M3**

---

### Pedidos y Ventas

- Como mayorista, quiero ver los pedidos realizados de mis productos para gestionarlos correctamente.  
  → **M2, M3**

- Como mayorista, quiero actualizar el estado del pedido (pendiente, aprobado, enviado, entregado).  
  → **M2, M3**

- Como mayorista, quiero ver un historial de ventas para analizar el desempeño de mi negocio.  
  → **M3**

---

### Reputación y Relación con Clientes

- Como mayorista, quiero recibir reseñas de clientes verificados para fortalecer mi reputación.  
  → **M3**

- Como mayorista, quiero responder reseñas de clientes para mejorar la comunicación y confianza.  
  → **M3**

---

### Cuenta y Control

- Como mayorista, quiero iniciar y cerrar sesión para acceder a mi cuenta de forma segura.  
  → **M0 (iniciar), M1, M2, M3 (iniciar y cerrar)**

- Como mayorista, quiero eliminar mi cuenta para controlar mis datos.  
  → **M1, M2, M3 (tener reglas definidas)**
